Because of this, C shares a lot of similar traits with Java, who is also a descendant from the C family. C is designed to run as .NET apps, this means that C Sharp has a plethora of uses. Origns

C sharp was designed by Microsoft employee Anders Heklsberg in the year 2000. C# was created as a general use coding language and developed to be used with the newly created .NET framework. At the time of release, C# and Java were great competitors in the world of computer languages. They both borrowed a lot from each other. Since C# was developed my Microsoft, it became quite common to see a windows application using C#. C# was also useful in building websites as well.
